LinkRouter

Route links to the right browser and profile

A Windows app that intelligently routes URLs from Teams, Outlook, Slack, and other apps to the correct browser and profile based on your rules.

Download for Windows View on GitHub

Windows 10/11 (64-bit) • Free & Open Source

Features

🌐

Multi-Browser Support

Works with Chrome, Edge, Firefox, Brave, Opera, and Opera GX with full profile support.

👤

Profile Aware

Automatically detects all your browser profiles (personal, work, dev) and routes accordingly.

📦

URL Groups

Group related domains together. Built-in templates for Microsoft 365 and Google Workspace.

📋

Clipboard Monitoring

Optionally monitor clipboard for URLs and route them with a single click.

🔔

Smart Notifications

Toast notifications let you quickly create rules for new domains.

💾

Backup & Restore

Export and import your configuration. Automatic rolling backups keep your data safe.

How It Works

1

Install & Set as Default

Install LinkRouter and set it as your default browser in Windows Settings.

2

Create Rules

Add rules for domains you want to route to specific browsers and profiles.

3

Click Links

Links from any app are automatically routed to the right browser.
